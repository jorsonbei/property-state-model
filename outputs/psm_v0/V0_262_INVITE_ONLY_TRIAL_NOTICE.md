# PSM V0.262 邀请制外部试用告知

## 试用性质

这是一个由项目操作员现场监督的研究型 Alpha 试用，不是公开服务，也不提供医疗、法律、投资、交易或其他专业决定。

## 参与范围

- 仅限 3 至 5 名受邀成年人。
- 参与者使用本地化名编号，不登记姓名、邮箱、电话或证件信息。
- 操作员必须先在试用系统之外确认受邀者为成年人，再用本机秘密 HMAC 将该受邀者与化名一对一绑定；参与者自行勾选成年不足以通过。
- 不开放匿名注册、公开链接或无人监督的远程访问。

## 请勿输入

- 姓名、联系方式、精确地址、账号、密码、API 密钥或支付资料；
- 病历、个人诊断、真实法律案件、券商账户或个人投资决定；
- 未成年人资料、生物识别资料、私人文件或未公开文稿；
- 任何自己无权提交的资料。

检测到上述内容时，系统和操作员应立即中止当前请求；该内容不得用于训练、规则更新或外部模型调用。

## 数据处理

- 原始问题只在当前处理和浏览器会话中短暂存在，服务器不保存原文。
- 项目最多保存本地化名、同意凭据、带本机秘密盐的消息 HMAC、风险类别、延迟、token 数和不含内容的事件记录。
- 参与者聊天内容不会提交给 OpenAI 或其他外部 API。外部 API 只可评审合成协议和合成安全样本。
- 运行元数据最长保存 7 天，期满自动删除；退出试用时立即停止新处理并触发删除。

## 风险与停止

模型可能答错、遗漏或给出不适合实际采用的建议。出现敏感资料、关键事实错误、安全漏判、预算越界、公开网络暴露、删除失败或安全事件时，试用必须停止，且不能自动恢复。

## 同意

操作员必须先展示本告知书，记录展示时间；参与者确认后记录确认时间，再记录明确同意时间。只有三个时间顺序有效且操作员已完成成年核验与受邀绑定，系统才能启用会话和第一条消息。

开始前，参与者需要确认：

1. 自己已满 18 岁并为受邀参与者；
2. 已阅读本告知并理解这不是专业服务；
3. 不会提交被禁止的资料；
4. 同意上述短期、最小化处理方式；
5. 知道可以随时退出并要求删除。

未完成明确同意时，不得开始试用。
