# Security Policy

This repository is an experimental local alpha. Do not expose the included HTTP server directly to the public internet. It has no authentication, tenant isolation, rate limiting, or production secret-management layer.

The former invite-only enrollment and trial endpoints were retired in V0.295 and return HTTP 410. Historical modules remain for audit and must not be treated as an active participant workflow.

Do not submit real medical records, legal matters, brokerage credentials, API keys, unpublished manuscripts, or other sensitive data in issues or pull requests.

For a private vulnerability report, use GitHub's private vulnerability reporting feature after the repository is published.
